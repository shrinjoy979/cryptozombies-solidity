# 🧟 CryptoZombies Solidity

A Solidity-based smart contract project inspired by CryptoZombies for learning Ethereum smart contract development in a fun and practical way.

This project demonstrates how to build zombie-themed smart contracts using Solidity, covering core blockchain concepts such as state variables, functions, structs, mappings, inheritance, and ownership.

---

## 🚀 Features

- Create and manage zombies on-chain  
- Learn Solidity through practical examples  
- Use structs and mappings for data storage  
- Generate zombie DNA  
- Ownership-based zombie control  
- Basic smart contract game logic  
- Beginner-friendly Solidity project structure  

---

## 🛠 Tech Stack

- Solidity  
- Ethereum  
- Remix / Hardhat / Truffle  
- JavaScript
